# Introduction to Utilities

This section contains various tools and scripts to help you automate or enhance your workflows across SAS Viya. Below you find a quick introduction of them.

## [SAS Job Definition Overview](./job-definitions-index.md)

This utility collects information about all of the SAS Job Definitions in an environment, turns it into three tables and also provides a template SAS Visual Analytics report that can be used to easily understand your environment.
